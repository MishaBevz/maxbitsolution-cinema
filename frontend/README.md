## Установка и запуск frontend

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Запустить dev‑сервер:
   ```bash
   npm run dev
   ```

Приложение будет доступно по адресу `http://localhost:5173`.

3. Запустить тесты:
   ```bash
   npm run test
   ```

## Подключение к бэкенду

Перед запуском фронтенда убедитесь, что запущен backend на `http://localhost:3022`  
(инструкция по запуску находится в `../backend/README.md`).
